# Project Documentation

## Overview
This repository contains the wholesale e-commerce platform built according to the PRD plan.

## Quick Links
- [`plan.md/`](../plan.md/plan.md) - Full Product Requirements Document (PRD) with system flow, architecture, and development phases
- [`docs/`](./README.md) - Project documentation and technical references

## Getting Started
1. Review the PRD in [`plan.md/plan.md`](../plan.md/plan.md) for complete project details
2. Follow the Development Priority Order (Phase 1 → Phase 2 → Phase 3)
3. Keep modules independent as per AI-Friendly Development Rules

## Development Phases
- **Phase 1**: Core Commerce (Auth, Users, Products, Categories, Cart, Orders, Payments)
- **Phase 2**: Wholesale Features (Tier Pricing, RFQ, Bulk Orders, Vendor System, Inventory)
- **Phase 3**: Engagement Features (Loyalty, Reviews, Digital Catalogs, Notifications)
